# Bliss Engine

Bliss Engine is a versatile open-source game engine capable of creating both 2D and 3D games. Designed using Java and C/C++, and leveraging the power of OpenGL and LWJGL (Lightweight Java Game Library) 2.9, Bliss Engine provides developers with a robust platform for game development.

## Features

- **2D and 3D Game Development**: Supports both 2D and 3D game creation to cater to a diverse range of projects.
- **Cross-Platform**: Engineered to run on multiple platforms, ensuring your games can reach a wider audience.
- **OpenGL Integration**: Harnesses the full power of OpenGL for graphics rendering to deliver stunning visual experiences.
- **LWJGL 2.9 Support**: Utilizes LWJGL 2.9 for efficient game loop and system management, input handling, and more.
- **Extensible Architecture**: Designed with extensibility in mind, allowing for easy expansions and customizations.

### Prerequisites

- Java Development Kit (JDK) 8 or newer
- C/C++ Compiler (GCC for Linux/Mac, MSVC for Windows)
- Integrated Development Environment (IDE) of your choice (Eclipse, IntelliJ IDEA, etc.)
- OpenGL drivers installed on your machine
